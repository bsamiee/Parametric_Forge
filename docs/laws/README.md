# [LAWS_CORPUS]

Repo-wide design and maintenance law: the landing doctrine, the universal design law with its machine-surface extensions, the coupling topology every cross-surface edit obeys, the pattern rows binding more than one language branch, and the scar ledger of regression-proven laws. Heavy workflow runs (`.claude/workflows/`) land their generalizable findings here through one terminal doctrine stage; the registry pages are the system of record only for facts no other owner carries.

A landing hardens an existing row before extending a page, and extends a page before minting one; a new page requires a law family no existing page charters. A fact owned elsewhere never gains a copy here — the owner keeps it, and a topology row may name it as a counterpart. A row earns admission by recurrence across independent units or by a structural singleton whose blast radius spans branches; the landing stage records that evidence in its run report, and the row itself states only the law. `landing.md` owns the full landing bar, per-surface justification, and page grammar.
