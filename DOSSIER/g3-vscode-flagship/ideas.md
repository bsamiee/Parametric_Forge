## [01]-[CLOSED_GAPS]

Facts closing the wave-1 GAPS, all live-verified.

- Keybindings write path (capabilities gap 1): Home Manager writes `keybindings.json` as a read-only Nix-store symlink, wholesale — `"${keybindingsFilePath n}".source = ... jsonFormat.generate "vscode-keybindings" (map (lib.filterAttrs (_: v: v != null)) v.keybindings)` with `jsonFormat = pkgs.formats.json { }`; no merge, no `onChange` (the module's only `onChange` is the extensions block). Hand-edits cannot survive a switch, and the file is unwritable between switches. Source: https://raw.githubusercontent.com/nix-community/home-manager/master/modules/programs/vscode/mkVscodeModule.nix (write block lines 417-423, `jsonFormat` line 32, path derivation lines 34-47).

Keyboard shortcuts sync as their own resource, scoped per-platform by `settingsSync.keybindingsPerPlatform`. Source: https://code.visualstudio.com/docs/configure/settings-sync; https://raw.githubusercontent.com/microsoft/vscode/main/src/vs/platform/userDataSync/common/userDataSync.ts.
